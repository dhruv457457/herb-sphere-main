# Virtual Herbal Garden

## Overview
The **Virtual Herbal Garden** is an interactive and educational web application developed by **Team Ayurhub** for the **Smart India Hackathon (SIH) 2024**. This project showcases a diverse range of medicinal plants used in AYUSH (Ayurveda, Yoga & Naturopathy, Unani, Siddha, and Homeopathy), aiming to provide an immersive learning experience about the benefits and uses of various herbs.

## Features
- **Interactive 3D Models**: View and interact with detailed 3D models of plants.
- **Augmented Reality (AR) Integration**: Explore plants in your environment using AR.
- **Search and Filter Options**: Easily find specific herbs and filter based on criteria.
- **Detailed Plant Information**: Learn about each herb's medicinal properties and uses.
- **Multimedia Integration**: Includes images, videos, and audio descriptions of plants.
- **Chatbot Assistance**: Get instant help and information about herbs.
- **Bookmarking and Note-taking**: Save your favorite plants and take notes for future reference.

## Tech Stack
- **Frontend**: React.js, Tailwind CSS
- **3D Modeling**: Blender, Sketchfab, WebGL
- **AR Features**: AR.js, three.js
- **Backend**: Node.js, MongoDB
- **Chatbot**: Dialogflow
- **Image Recognition**: Plant.id API
- **Hosting**: AWS S3
